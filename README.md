## 🌟 Overview
**Facebook Custom ChatBot** is more than just an auto-reply chatbot. It is a complete mini-ecosystem on Messenger that allows users to manage their personal finances through a built-in **Wallet**, while providing admins with a powerful **real-time Monitoring** system to track all activities.

---

## 🔥 Key Features

### 💳 Smart Wallet
*   **Messaging-based Transactions:** Transfer money, top up directly via Facebook Messenger messages.
*   **Transaction History:** View history as charts or detailed lists.
*   **Security:** End-to-end encryption for balance inquiries.
*   **Integrations:** Supports popular payment gateways (Momo, VNPay, Crypto).

### 📊 Advanced Monitoring
*   **Real-time Analytics:** Track message volume, new users, and activity every second.
*   **Health Check:** Automatic alerts when the system encounters errors or high latency.
*   **User Behavior Analysis:** AI-powered analysis of user behavior to provide personalized suggestions.

### 🤖 Chatbot Core
*   **NLP Processing:** Natural language understanding (Vietnamese/English).
*   **Custom Commands:** Highly flexible custom command creation.
*   **Smart Auto-Reply:** Context-aware intelligent responses.

## 🌟 Tổng quan
**ChatBot Facebook Custom** không chỉ là một chatbot trả lời tự động. Đây là một hệ sinh thái thu nhỏ trên Messenger, cho phép người dùng quản lý tài chính cá nhân thông qua **Wallet**, đồng thời cung cấp cho admin một hệ thống **Monitoring** mạnh mẽ để theo dõi mọi hoạt động theo thời gian thực.

## 🔥 Tính năng nổi bật

### 💳 Ví thông minh (Smart Wallet)
*   **Giao dịch bằng tin nhắn qua messenger FB:** Chuyển tiền, nạp tiền trực tiếp qua tin nhắn.
*   **Lịch sử giao dịch:** Truy xuất lịch sử dưới dạng biểu đồ hoặc danh sách chi tiết.
*   **Bảo mật:** Hệ thống mã hóa đầu cuối cho các truy vấn số dư.
*   **Tích hợp:** Hỗ trợ cổng thanh toán (Momo, VNPay, Crypto).

### 📊 Giám sát tối thượng (Advanced Monitoring)
*   **Real-time Analytics:** Theo dõi số lượng tin nhắn, người dùng mới mỗi giây.
*   **Health Check:** Tự động cảnh báo khi hệ thống gặp lỗi hoặc latency cao.
*   **User Behavior:** Phân tích hành vi người dùng bằng AI để đưa ra gợi ý phù hợp.

### 🤖 Chatbot Core
*   **NLP Processing:** Hiểu ngôn ngữ tự nhiên (Vietnamese/English).
*   **Custom Commands:** Tạo lệnh tùy chỉnh linh hoạt.
*   **Auto-Reply:** Phản hồi thông minh dựa trên ngữ cảnh.
